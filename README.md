## Php Type Resolver

[TypeResolver](src/main/java/me/pesekjak/phptr/TypeResolver.java) that determines
the type of a PHP variable based only on its `@var` documentation block.

### Building and Testing
**The project is built on Java 25 and uses Java 25 features. The build will
not work on earlier versions.**

To run the test suite run
```
./gradlew test
```

To build the project run
```
./gradlew build
```
